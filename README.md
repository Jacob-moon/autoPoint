# autoPoint

ZEP 포인트 지급기를 여러 명 대상 일괄 지급으로 자동화하는 로컬 툴입니다.

## 기능
- 이름/포인트 목록을 한 번에 입력해 순차 지급
- 지급 정책(주간/우수 TIL/학습 몰입 등) 프리셋 제공
- 지급 후 채팅 메시지 자동 입력(선택)
- Google 로그인 계정 고려: Playwright 영구 프로필(`.auth`)로 로그인 세션 재사용

## 실행 방법
```bash
npm install
npx playwright install chromium
npm start
```

브라우저에서 `http://localhost:3000` 접속 후 사용합니다.

## 입력 형식
대상자 목록 입력창에 아래처럼 한 줄씩 작성:

```text
홍길동,400
김코딩,1000
이수강
```

- 포인트 생략 시 기본 지급 포인트 사용.

## 주의 사항
- ZEP 위젯 DOM 구조가 변경되면 CSS 선택자 입력칸에 직접 selector를 넣어 보정하세요.
- 첫 실행 때 열리는 브라우저에서 Google 로그인/권한 확인이 필요할 수 있습니다.
